
# Image Similarity App

A Flutter application that compares two images and checks whether they are similar.

## Features

* Select two images.
* Compare image similarity.
* Display **"تمت المهمة بنجاح"** when the images are similar.
* Display **"حاول مرة أخرى"** when the images are not similar.
* Simple and responsive user interface.

## Technologies

* Flutter
* Dart

## Getting Started

1. Clone the repository:

```bash
git clone <repository-url>
```

2. Install dependencies:

```bash
flutter pub get
```

3. Run the application:

```bash
flutter run
```

## Project Goal

The purpose of this project is to demonstrate image comparison functionality in Flutter and provide visual feedback based on the similarity result.
